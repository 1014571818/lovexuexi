## 使用说明

- 文章和视频：点击“切到 强国”，在强国主界面点击“开始浏览”，即可自动浏览，**并且会记录已阅读的文章和视频到数据库（不同账号单独记录，解决多账号问题；自动清除7天以上记录）**，挑战答题、双人对战和每日答题自动完成，集满全部得分结束。（部分手机停在"获取学号"，点击"切到强国"，可解决）
- 挑战答题：进入挑战答题界面，点击“挑战答题”，开始答题，答错会自动重新开始，得到6分停止。
- 争上游：在强国主界面，点击“争上游”，自动答题两轮，返回主界面。（因对战的不确定性，暂时未加入自动学习中。）
- 每日答题：进入每日、专项、每周界面，点击"每日答题等"。程序会自行进行答题，得到6分停止。
- 手动改题：手动查改删增，支持以题目或答案关键字查询，或列出最后十条入库记录
- 题库：由于网络题库的不确定性，将逐步摒弃网络题库。通过挑战答题，自主更新题库。

本项目仅供autojs交流学习，请勿用于商业
